# BFC


是页面盒模型中的一种 `CSS` 渲染模式

相当于一个独立的容器，里面的元素和外部的元素相互不影响。


::: info
块级格式化上下文，是web页面css渲染的一部分，决定了自元素的排列和其他元素的关系。垂直方向从上而下的排列，两个上下的BFC的容器margin会重叠，浏览器因为四舍五入造成多列布局换行
:::





### 创建 `BFC` 的方式有：

1. `html` 根元素

2. `float` 浮动

3. 绝对定位

4. `overflow` 不为 `visible`

5. `display` 为表格布局或者弹性布局；





`BFC` 主要的作用是：

6. 清除浮动

7. 防止同一 `BFC` 容器中的相邻元素间的外边距重叠问题

# 闭包

闭包让开发者可以从内部函数访问外部函数的作用域。

### 特征是
1. 是一个函数
2. 能访问另一个函数作用域中的变量

闭包很有用，因为它允许将函数与其所操作的某些数据（环境）关联起来。这显然类似于面向对象编程。


### 用途
1. **私有变量**：闭包可以用于创建`私有变量`，因为闭包中的变量在函数外部是不可访问的。
2. **回调函数**：闭包经常用于创建回调函数，使得回调函数能够访问其被创建时的上下文。

# 箭头函数



## 特征
- 箭头函数没有独立的 this、arguments 和 super 绑定，并且不可被用作方法。
- 箭头函数不能用作构造函数。使用 new 调用它们会引发 TypeError。
- 箭头函数不能在其主体中使用 yield，也不能作为生成器函数创建。
- 不能通过call, apply, bind修改this指向。


::: info
箭头函数的this永远指向父级作用域
:::
